Forms


El elemento "form" en HTML define un formulario que se utiliza para recopilar una entrada del usuario:
Un formulario HTML contiene elementos de formulario . Los elementos de formulario son diferentes tipos de elementos de entrada, como:
campos de texto, casillas de verificación, botones de opción, botones de envío y más.

Input

Es el elemento más importante de los forms, y puede ser de distintos tipos: texto, radio, y submit

Label

define una etiqueta para muchos elementos de formulario. Es útil para los usuarios de lectores de pantalla, porque el lector de pantalla leerá la etiqueta cuando el usuario se centre en el elemento de entrada.

Botón Submit

El controlador de formulario suele ser un archivo en el servidor con un script para procesar datos de entrada.

El controlador de formulario se especifica en el atributo del formulario.action





Atributos

Action

El atributo "action" define la acción que se va a realizar cuando se envía el formulario
Por lo general, los datos del formulario se envían a un archivo en el servidor cuando el usuario hace clic en el botón enviar.

Target

El atributo especifica dónde mostrar la respuesta que se recibe después de enviar el formulario.
Puede tener los valores de: _bank, _top, _self, _parent. El valor predeterminado es _self
Comprobar los target

Method

El atributo especifica el método HTTP que se utilizará al enviar los datos del formulario Los datos del formulario se pueden enviar como variables URL (get) o como transacción HTTP (post) El método HTTP predeterminado al enviar datos de formulario es GET.





  • Anexa los datos del formulario a la dirección URL, en pares nombre/valor
  • ¡NUNCA use GET para enviar datos confidenciales! (¡los datos del formulario enviados son visibles en la URL!)
  • La longitud de una URL es limitada (2048 caracteres)
  • Útil para envíos de formularios en los que un usuario desea marcar el resultado
  • GET es bueno para datos no seguros, como cadenas de consulta en Google





  • Anexa los datos del formulario dentro del cuerpo de la solicitud HTTP (los datos del formulario enviados no se muestran en la dirección URL
  • POST no tiene limitaciones de tamaño y se puede utilizar para enviar grandes cantidades de datos.
  • Los envíos de formularios con POST no se pueden marcar como favoritos

Autocompletar

El atributo especifica si un formulario debe tener autocompletado encendido o desactivado.
Cuando el autocompletado está encendido, el navegador completa automáticamente los valores en función de los valores que el usuario ha introducido antes.

Prueba poniendo el autocompletado en "off"

Tipos de Atributos

Value

el atributo "value" valor inicial para un campo de entrada




Readonly

el atributo "readonly" especifica que un campo de entrada es de solo lectura




Disabled

el atributo "Disabled" especifica que se debe deshabilitar un campo de entrada




Size

el atributo "Size" especifica el ancho visible, en caracteres, de un campo de entrada


Nota:El atributo funciona con los siguientes tipos de entrada: texto, búsqueda, tel, url, correo electrónico y contraseña.



Maxlenght

el atributo "Maxlenght" especifica el número máximo de caracteres permitidos en un campo de entrada.


Nota:Cuando se establece este atributo, el campo de entrada no aceptará más del número de caracteres especificado. Sin embargo, este atributo no proporciona ningún comentario. Entonces, si desea alertar al usuario, debe escribir código JavaScript.



Min & Max

el atributo "Min o Max" especifican los valores mínimo y máximo para un campo de entrada.


Los atributos y funcionan con los siguientes inputs: número, rango, fecha, fecha-hora-local, mes, hora y semana.


Utilice los atributos max y min juntos para crear un rango de valores legales.




Multiple

el atributo "Multiple" especifica que el usuario puede introducir más de un valor en un campo de entrada

El atributo funciona con los siguientes tipos de entrada: correo electrónico y archivo
Selecciona varios archvios matentiendo CTRL o Shift mientas seleccionas.

Pattern

el atributo "Pattern" especifica una expresión regular con la que se comprueba el valor del campo de entrada cuando se envía el formulario

El atributo funciona con los siguientes tipos de entrada: texto, fecha, búsqueda, url, tel, correo electrónico y contraseña
Consejo: Utilice el atributo title global para describir el patrón que ayudará al usuario.
Consejo: Obtén más información sobre las expresiones regulares de JavaScrip

Placeholder

Especifica una breve sugerencia que describe el valor esperado de un campo de entrada

El atributo funciona con los siguientes tipos de entrada: texto, búsqueda, url, tel, correo electrónico y contraseña.

Required

Especifica que se debe rellenar un campo de entrada antes de enviar el formulario

El atributo funciona con los siguientes tipos de entrada: texto, búsqueda, url, tel, correo electrónico, contraseña, selectores de fecha, número, casilla de verificación, radio y archivo.

Step

Especifica los intervalos de números legales para un campo de entrada

Ejemplo: si step="3", los números legales podrían ser -3, 0, 3, 6, etc.
El atributo funciona con los siguientes tipos de entrada: número, rango, fecha, fecha-hora local, mes, hora y semana
Consejo: Este atributo se puede utilizar junto con los atributos max y min para crear un rango de valores legales.
Nota:Las restricciones de entrada no son infalibles, y JavaScript proporciona muchas formas de agregar entradas ilegales. Para restringir de forma segura la entrada, ¡también debe ser verificada por el receptor (el servidor)!

Height & Width

La entrada y los atributos especifican el alto y el ancho de un elemento, únicamente de una imagen o frame de video

Consejo:Especifique siempre los atributos de altura y anchura de las imágenes. Si se establecen la altura y la anchura, el espacio necesario para la imagen se reserva cuando se carga la página. Sin estos atributos, el navegador no conoce el tamaño de la imagen y no puede reservar el espacio adecuado para ella. El efecto será que el diseño de la página cambiará durante la carga (mientras se cargan las imágenes).

El atributo funciona con y los siguientes tipos: texto, búsqueda, url, tel, correo electrónico, contraseña, selectores de fecha, rango y color.
