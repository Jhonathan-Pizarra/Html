Multemedia

Las páginas web a menudo contienen elementos multimedia de diferentes tipos y formatos

Los elementos multimedia (como audio o video) se almacenan en archivos multimedia.
La forma más común de descubrir el tipo de archivo es mirar la extensión del archivo.
Los archivos multimedia tienen formatos y diferentes extensiones como: .wav, .mp3, .mp4, .mpg, .wmv y .avi.

Formatos de video

Hay muchos formatos de video por ahí. Los formatos MP4, WebM y Ogg son compatibles con HTML.
El formato MP4 es recomendado por YouTube.

Formatos de audio

Los formatos de audio MP3, WAV y Ogg son compatibles con el estándar HTML.
MP3 es el mejor formato para música grabada comprimida. El término MP3 se ha convertido en sinónimo de música digital.
Si su sitio web se trata de música grabada, MP3 es la opción.

Geolocalización

La API de geolocalización HTML se utiliza para obtener la posición geográfica de un usuario.
La geolocalización es más precisa para dispositivos con GPS, como teléfonos inteligentes.
Dado que esto puede comprometer la privacidad, la posición no está disponible a menos que el usuario la apruebe.

A partir de Chrome 5.0, la API de geolocalización solo funcionará en contextos seguros como HTTPS.
Si su sitio está alojado en un origen no seguro (como HTTP), las solicitudes para obtener la ubicación de los usuarios ya no funcionarán.
